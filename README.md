# Carrefour AI Nutrition Ecosystem - A/B Testing Project

## Project Overview
This project is a high-fidelity landing page and interactive A/B testing showcase designed for the **Carrefour GreenMeal** student project. It demonstrates our digital strategy to combat inflation and food waste through AI-driven nutrition planning.

## Features
- **Product Landing Page**: A modern, responsive showcase of the GreenMeal app features (AI Planner, EcoPoints, Anti-Waste).
- **Interactive A/B Testing Lab**: A functional dashboard demonstrating 10 specific hypothesis-driven experiments across the customer journey.
- **Simulation**: Users can click to simulate user actions and see real-time "results" of our proposed tests.

## Customer Journey Stages Optimized
1.  **Awareness**: Ad creative & value proposition testing.
2.  **Consideration**: App Store optimization.
3.  **Decision**: Onboarding flow friction reduction.
4.  **Use**: Notification tone & cross-selling.
5.  **Loyalty**: Reward framing & viral loops.
